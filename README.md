# GhorerBajar Shop Design By AI

This project is a modern ecommerce website concept for a Bangladeshi startup brand.  
The website was designed with AI using a single structured prompt focused on creating a clean, trustworthy, and mobile-first shopping experience for Bangladesh.

## Prompt
```text
Design and generate a complete modern ecommerce website for a Bangladeshi startup brand selling household and lifestyle products. The website should feel trustworthy, clean, conversion-focused, and visually premium, while still being friendly and accessible for everyday Bangladeshi shoppers.

Create a Bangladesh-focused ecommerce experience for products such as household items, small electric appliances, gadgets, accessories, kitchen products, and selected personal care items.

Use a mixed Bangla + English interface:
- Main navigation and menu items in English
- Most banners, product highlights, descriptions, checkout labels, delivery information, and CTA text in Bangla

The design should feel:
- clean
- warm
- organized
- modern
- startup-friendly
- high-conversion

Use a mobile-first ecommerce layout with:
- homepage hero section
- category highlights
- top-selling products
- promotional offer sections
- product listing pages
- product details pages
- simple Bangladesh-friendly checkout
- about and contact pages
- clean footer links

The visual style should use soft light backgrounds, deep green or dark neutral tones, clear typography, polished product cards, strong CTA buttons, and generous spacing.

The final website should look practical, beautiful, and trustworthy for real online shoppers in Bangladesh.
```

## Conclusion

This design was generated from a single prompt in z.ai, which made the website creation process much faster and more accessible. Instead of manually planning every layout block and visual decision from scratch, the prompt provided a strong creative direction for a Bangladeshi ecommerce store with mixed Bangla and English content.
